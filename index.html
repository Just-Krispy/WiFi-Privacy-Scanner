🎯 What This Tool Does

WiFi Privacy Scanner is an open-source security tool that scans nearby WiFi networks and identifies routers capable of advanced surveillance techniques using Channel State Information (CSI).

🎯 Motion Sensing

Detects movement and presence through walls without cameras

💓 Vital Signs Monitoring

Can detect heart rate, breathing patterns, and other biometrics

🚶 Biometric Tracking

95.5% accuracy identifying individuals by their WiFi "signature"

🏃 Pose Estimation

Full body tracking and activity recognition through solid objects

⚠️ Known Threat Systems

The scanner detects these surveillance-capable systems:

  • 🔴 Xfinity (XB6/XB7/XB8) - Motion sensing and occupancy tracking
  • 🔴 Origin AI - Vital signs monitoring and CSI extraction
  • 🔴 WhoFi - 95.5% biometric identification accuracy
  • 🔴 DensePose - Full body pose estimation
  • 🔴 Google Nest - Soli radar-based presence detection
  • 🔴 Cognitive Systems - WiFi motion detection platform

📊 Risk Levels

Each network is scored 0-100 and categorized into risk levels:

🔴 CRITICAL (80-100) Full CSI extraction, biometric tracking - AVOID
🟠 HIGH (60-79) Motion sensing enabled - DISABLE OR REPLACE
🟡 MEDIUM (40-59) Basic presence detection - REVIEW SETTINGS
🟢 LOW (0-39) Standard WiFi router - SAFE

🚀 Usage Instructions

1. Installation

No external dependencies required - uses Python 3 standard library:

git clone https://github.com/your-repo/wifi-privacy-scanner.git
cd wifi-privacy-scanner

2. Run a Scan

python3 scripts/scanner.py

The scanner will:

  • Detect all nearby WiFi networks
  • Analyze each router against the CSI threat database
  • Calculate a threat score (0-100)
  • Save results to output/scan_*.json

3. Generate Reports

LATEST=$(ls -t output/scan_*.json | head -1)
python3 scripts/report_generator.py "$LATEST"

Creates human-readable reports in Markdown and HTML formats.

4. View Results

# View latest scan summary
cat output/scan_*_report.md

# Or open the HTML report in a browser
firefox output/scan_*_report.html

🛡️ Privacy Protection Tips

  • ✅ Enable MAC Randomization
    Prevents device tracking across networks
  • 🔒 Use a VPN
    Encrypts all traffic, even on untrusted networks
  • 📴 Disable WiFi When Not Needed
    Limits exposure to surveillance capabilities
  • 🔧 Replace CSI-Capable Routers
    Use standard routers without motion/presence features
  • ⚙️ Review Router Settings
    Disable motion detection, presence sensing, and remote management
